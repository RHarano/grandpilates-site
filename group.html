ABOUT

最大8名・1人1台の少人数クラス

グループレッスンは、リフォーマーとタワーを使用した60分のクラスです。 最大8名までの少人数制で、お一人につき1台のマシンをご用意。 資格を持つインストラクターがレベルに合わせて丁寧に指導するので、初めての方も安心してご参加いただけます。

REFORMER CLASS

リフォーマークラス

腹筋・背中・肩・お尻・骨盤底筋を強化し、体幹を鍛えるクラスです。姿勢の改善、ケガの予防、膝・股関節・腰の関節痛の軽減にも効果が期待できます。

BASIC 1基礎エクササイズを丁寧に。初めての方はこちらから。
BASIC 1.5Basic 1より少しチャレンジングに、テンポよく。
FLOW 1.5基礎の上に応用エクササイズを積み重ねます。
FLOW 2バランスとコントロールを重視したハイテンポクラス。
TOWER CLASS

タワークラス

スプリングとバーを使い、リハビリ由来の丁寧な動きで、左右のバランス・背骨の柔軟性・深層筋の活性化を促します。ご年配の方や、ケガからの回復期の方にもおすすめです。

BASIC 1安全に、基礎からしっかりと。
BASIC 1.5強度を上げてステップアップ。

※ 初めてご参加の方は「Basic 1」または「Basic 1.5」からのスタートをおすすめしています。
